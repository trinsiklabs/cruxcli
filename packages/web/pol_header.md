# Political Belief Decomposition — Layer 1: Atomic Beliefs

**Purpose:** Identify every independently assessable belief that, bundled together, form political ideologies. This is the master catalog from which individual system pages, historical implementation pages, and assessment questions are built.

**Method:** Each belief is stated once, tagged with every system that holds it, its underlying presupposition about human nature/society, a neutral assessment phrasing, counter-belief, research citations, and confidence rating.
